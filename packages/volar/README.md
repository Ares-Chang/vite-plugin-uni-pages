# @uni-helper/volar-plugin-uni-pages

为 `<route>` 块 提供 IntelliSense

## 安装

```bash
pnpm add -D @uni-helper/volar-plugin-uni-pages
```

## 用法

```js
// volar.config.js
const volarPluginUniPages = require('@uni-helper/volar-plugin-uni-pages')

module.exports = {
  plugins: [
    volarPluginUniPages(),
  ],
}
```
